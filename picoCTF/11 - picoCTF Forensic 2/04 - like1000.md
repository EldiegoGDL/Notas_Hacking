## Descripción
This [.tar file](https://jupiter.challenges.picoctf.org/static/52084b5ad360b25f9af83933114324e0/1000.tar) got tarred a lot.
## Solución
con wget descargamos el archivo despues usamos file para ver que tipo de archivo es
```
┌──(diego㉿Maquina-deGuerra)-[~/Documentos/forensic/like1000]
└─$ file 1000.tar 
1000.tar: POSIX tar archive (GNU)
                                   
```
al ver que se trata de un archivo .tar vemos el contenido sin descomprimir, 
```
┌──(diego㉿Maquina-deGuerra)-[~/Documentos/forensic/like1000]
└─$ tar -tf 1000.tar
999.tar
filler.txt

```
vemos que es un multiple tar con 1000 capas para descomprimirlo y no tener todas esas capas la descomprimimos y al mismo tiempo borramos las capas
```
┌──(diego㉿Maquina-deGuerra)-[~/Documentos/forensic/like1000]
└─$ for i in {1000..1}; do tar -xf $i.tar && rm $i.tar ; done
┌──(diego㉿Maquina-deGuerra)-[~/Documentos/forensic/like1000]
└─$ ls
filler.txt  flag.png

```
luego revisamos el contenido de los archivos que obtuvimos
```
┌──(diego㉿Maquina-deGuerra)-[~/Documentos/forensic/like1000]
└─$ cat filler.txt
alkfdslkjf;lkjfdsa;lkjfdsa

```
